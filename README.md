1. Project Overview
Waleizy Tech NPO is a student-led non-profit organisation established in 2026 by IIE students in Bloemfontein. The website supports small business owners who cannot afford web developers. Target users are spaza shops, salons, plumbers, photographers and other micro-businesses in Bloemfontein.  

The site promotes free 3-day HTML and CSS workshops, a laptop loan service at Bloemfontein City Library, and one-on-one help sessions with student volunteers. All hosting is done through GitHub Pages to keep costs at R0. The site is static, mobile-friendly, and written with HTML5 and CSS3 without frameworks.

2. Sitemap
